# Ethereal: Система агрегации и обработки распределенных данных о рыночной стоимости виртуальных предметов с применением алгоритмов машинного обучения

__________________________________________

### Цель проекта
Создать персонального аналитического ассистента, который освобождает пользователя от рутинного ручного мониторинга цен на различных торговых площадках. Система помогает находить наиболее выгодные моменты для обмена и покупки игровых предметов, используя алгоритмы прогнозирования и единую базу данных.

__________________________________________

### Описание идеи
Ethereal — это веб-сервис, который объединяет данные из нескольких независимых источников в один удобный интерфейс. Вместо того чтобы открывать пять вкладок и сравнивать цены вручную, пользователь получает готовую аналитику. 
Главная фишка проекта это **Deep Tracking**. Это не просто пассивный просмотр цен, а активная слежка за выбранными позициями. Наши алгоиртмы "знают" особенности каждой площадки, учитывают их скрытые комиссии и помогают пользователю увидеть реальную выгоду без калькулятора в руках.

__________________________________________

### Карта сайта (Sitemap)
Структура сервиса разделена на логические блоки для удобства навигации и разработки:
  **1. Информационный блок:**
    - **Главная (Landing):** Презентация преимуществ и принципа работы.
    - **Тарифы:** Описание уровней доступа (Free vs Deep Tracking).
    - **Поддержка/FAQ:** Обратная связь и заявки на добавление новых площадок.

  **2. Авторизация и профиль:**
    - **Вход / Регистрация:** Доступ к системе через подтвержденную почту.
    - **Настройки профиля:** Управление личными данными и активной подпиской.

  **3. Функциональное ядро (Рабочая зона):**
    - **Dashboard:** Общая сводка по рынку и быстрый доступ к активным трекам.
    - **Market Browser:** Каталог всех скинов, доступных на 5+ площадках одновременно.
    - **My Tracking:** Персональная страница со списком отслеживаемых предметов (слоты).
    - **Skin Detail Page:** Детальная карточка предмета с ценами всех площадок и историей.
    - **Deep Analytics:** Прогнозы нейросети и оценка рыночных рисков (Premium доступ).

__________________________________________

### MVP (Минимально жизнеспособный продукт)
На этапе демонстрации система будет выполнять базовый цикл работы:
  1.  **Управление доступом:** Регистрация и вход через электронную почту.
  2.  **Market Browser:** Просмотр единого каталога скинов, сопоставленного между площадками.
  3.  **Система трекинга:**
        - Добавление/удаление скина в активный слот отслеживания.
        - Отображение цен в реальном времени с 5 различных сайтов на одной странице.

### Направления развития (Future Vision)
- **Интеллектуальный прогноз:** Внедрение аналитической модели для предсказания трендов (рост/падение цены).
- **Внешние уведомления:** Мгновенные оповещения в Telegram при достижении целевой цены.
- **Система уровней:** Масштабирование количества слотов для трекинга (Deep Tracking).
- **Расширение базы:** Автоматизированная проверка и подключение новых добросовестных маркетов.
